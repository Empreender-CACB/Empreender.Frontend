
### Instalação



Instalação das dependências.

```bash
pnpm install
```

Server com hotreload (use para desenvolver).

```bash
pnpm run dev
```

### Lint

```bash
pnpm run lint
```

### Typecheck

```bash
pnpm run typecheck
```

### Build

```bash
pnpm run build
```

### Testes

```bash
pnpm run test
```
